These guides walk through the major runtime concepts in nx.js — from rendering
input from the gamepad and touchscreen, to saving data, to launching the
built-in web browser as a Library Applet. Each page focuses on _why_ you would
use a given API and shows a runnable example, with links into the API
reference for the full surface.
