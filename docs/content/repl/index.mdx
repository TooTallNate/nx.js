title: Usage
---

The `@nx.js/repl` package provides a REPL (read-eval-print loop) implementation.

It is specifically designed for use with nx.js applications, but is generic enough
that it can be used with arbitrary JavaScript runtimes.

## Installation

<InstallTabs items={['npm', 'pnpm', 'yarn', 'bun']}>
    <Tab value='npm'>

```bash
npm install @nx.js/repl
```

    </Tab>
    <Tab value="pnpm">

```bash
pnpm add @nx.js/repl
```

    </Tab>
    <Tab value='yarn'>

```bash
yarn add @nx.js/repl
```

    </Tab>
    <Tab value='bun'>

```bash
bun add @nx.js/repl
```

    </Tab>

</InstallTabs>
