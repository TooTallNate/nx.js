# Breakout

Classic [Breakout](<https://en.wikipedia.org/wiki/Breakout_(video_game)>) game implemented as an nx.js app for the Nintendo Switch.

Based on the MDN tutorial: [2D breakout game using pure JavaScript](https://developer.mozilla.org/en-US/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript).

## Controls

| Input | Action |
| --- | --- |
| D-pad / Left stick | Move paddle |
| Touchscreen | Drag to move paddle |
| A | Start / Restart |
| + | Pause / Exit |

## Features

- 5 rows × 8 columns of color-coded bricks
- 3 lives
- Score tracking
- FPS counter
- Touchscreen support
